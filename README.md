# DEMO DMII2

Demo project for Gobelins DMII2 Game Jam - Octobre 2022

### Setup the correct node version

```bash
nvm use
```

### Install dependencies

```bash
pnpm install
```

### Start working

```bash
pnpm run dev
```

### Build

```bash
pnpm run build
```

## Use static assets

Static assets should be place in the /public folder.
